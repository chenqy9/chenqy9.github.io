## 一、开启允许任何源

打开终端【启动台】-> 【其它】-> 【终端】，输入：

``` bash
sudo spctl  --master-disable
```

然后回车，继续输入密码，然后回车。

## 二、发现还是显示“已损坏，无法打开。 您应该将它移到废纸篓”

在终端输入以下命令，其中具体的软件路径替换成你想要打开的软件路径，按回车输入秘密即可。

``` bash
sudo xattr -r -d com.apple.quarantine /Applications/xxxx.app
```
