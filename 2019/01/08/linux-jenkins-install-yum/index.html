Linux安装升级jenkins

发表于 2019-01-08 更新于 2024-11-22 分类于 文章教程

本文介绍在 centos 操作系统下通过 yum 安装 jenkins，并通过下载官方 war 包的方式升级。

安装

  1. 安装 JDK

  2. 安装 Jenkins

  3. 启动 Jenkins 服务

  4. 注意事项

    Jenkins 默认端口是 8080，确保此端口不被占用或者修改 Jenkins 的默认端口，打开防火墙对该端口的拦截。

升级

  1. 下载官升升级 war 包，scp 到目标服务器

  2. 查看 Jenkins 服务启动的安装包位置
